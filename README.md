overview 
This project enhances the network infrastructure by extending connectivity to an external building and improving reliability through redundant switching. 
The solution ensures secure access to organizational systems while reducing downtime by eliminating single points of failure in the distribution layer.

**Project Objectives**

Extend network connectivity to an external office building

Provide secure and reliable access to organizational resources

Eliminate single points of failure in the distribution layer

Improve network fault tolerance and service availability

Minimize downtime and ensure operational continuity

**Technologies and Networking Concepts**

Virtual Local Area Networks (VLANs)

Inter-VLAN Routing

EtherChannel Link Aggregation

Spanning Tree Protocol (STP)

Redundant Switching Architecture

IP Addressing and Network Segmentation

Secure Remote Management (SSH)
